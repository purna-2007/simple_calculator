# Simple Calculator

Simple Calculator is a basic PHP project that performs fundamental arithmetic operations such as addition, subtraction, multiplication, and division. This project is designed for beginners to understand PHP logic and form handling in a simple way.

The project contains a single PHP file that processes user input and displays results instantly in the browser. It can be run on any local server environment that supports PHP such as XAMPP, WAMP, or Laragon.

## Features
- Perform basic arithmetic operations
- Lightweight and easy to understand
- Beginner-friendly PHP code

## Technologies Used
- PHP

## Project Structure
simple_calculator  
├── calculator.php  
└── README.md  

## Usage
Open the project on a local PHP server, enter two numbers, select the desired operation, and view the calculated result immediately.
